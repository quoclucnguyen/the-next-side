# Project Brief: The Next Side - Pantry Management App

## Project Name
**The Next Side** - Ứng dụng Quản lý Tủ bếp (Pantry Management Application)

## Project Goal
Phát triển một ứng dụng web mobile-first giúp người dùng theo dõi, quản lý và tối ưu hóa việc sử dụng thực phẩm trong tủ bếp của họ nhằm giảm lãng phí thực phẩm và tiết kiệm chi phí.

## Core Requirements

### Functional Requirements
1. **Pantry Management**: Theo dõi thực phẩm trong tủ lạnh, tủ đông và tủ khô
2. **Product Tracking**: Lưu trữ thông tin sản phẩm (tên, số lượng, hạn sử dụng, thương hiệu, giá)
3. **Expiration Management**: Theo dõi và cảnh báo hạn sử dụng
4. **Search & Filter**: Tìm kiếm và lọc sản phẩm theo loại, thời gian
5. **Recipe Suggestions**: Gợi ý công thức dựa trên nguyên liệu có sẵn
6. **Shopping List**: Tạo danh sách mua sắm tự động
7. **User Profiles**: Quản lý thông tin cá nhân và cài đặt

### Non-Functional Requirements
- **Performance**: Fast loading and smooth interactions on mobile devices
- **Responsive**: Works on various screen sizes (mobile-first approach)
- **Accessibility**: WCAG AA compliant
- **Scalability**: Support growing user base and product catalog
- **Reliability**: Consistent data synchronization and minimal downtime

## Target Users
- **Primary**: Người tiêu dùng cá nhân muốn quản lý thực phẩm trong gia đình
- **Secondary**: Người dùng quan tâm đến việc giảm lãng phí thực phẩm và tiết kiệm chi phí

## Success Metrics
- **Retention Rate**: Tỷ lệ người dùng quay lại sử dụng (target: 40%+ after 30 days)
- **Daily Active Users**: Số người dùng hoạt động hàng ngày
- **Food Waste Reduction**: Mức độ giảm lãng phí thực phẩm (target: 30%+)
- **Feature Adoption**: Tỷ lệ sử dụng các tính năng chính
- **User Satisfaction**: Điểm hài lòng của người dùng (target: 4.5/5)

## Technology Constraints
- **Frontend Framework**: React 19 with TypeScript
- **Styling**: Tailwind CSS 4.x
- **State Management**: Zustand
- **Routing**: React Router v7
- **UI Library**: Ant Design Mobile (antd-mobile)
- **Build Tool**: Vite
- **Testing**: Vitest + Playwright + Storybook
- **React Compiler**: Enabled for automatic optimization

## Project Scope
### Phase 1 (Current) - MVP Core Features
- Basic pantry list display
- Product details view
- Navigation between screens
- Basic component library
- Responsive layout

### Phase 2 - Data Persistence & Management
- Product CRUD operations
- Local storage integration
- Search and filtering functionality
- Expiration tracking

### Phase 3 - Advanced Features
- Barcode scanning
- Recipe integration
- Shopping list generation
- Push notifications

### Phase 4 - Backend & Cloud
- User authentication
- Cloud sync
- Analytics and reporting
- API integration

## Design Philosophy
- **Mobile-First**: Optimized for mobile experience, responsive for desktop
- **Clean & Simple**: Minimalist UI with focus on usability
- **Data-Driven**: Visual feedback and actionable insights
- **User-Centric**: Features designed around real user needs and behaviors
